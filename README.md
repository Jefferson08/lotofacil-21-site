# Lotofácil 21 — Landing Page

Landing page do Lotofácil 21 (Hotmart), construída com React + Vite + Tailwind + shadcn/ui.

## Requisitos
- Node.js 20+
- pnpm

## Rodar localmente
```bash
pnpm install
pnpm dev
```

## Build
```bash
pnpm build
pnpm preview
```

## Deploy (GitHub Pages)
**Importante:** para efetivar mudanças, sempre rode `pnpm run build` antes do deploy.

```bash
pnpm run build
pnpm run deploy
```

O deploy publica no branch `gh-pages`. Configure o GitHub Pages para usar esse branch.

## Scripts úteis
- `pnpm dev` — ambiente de desenvolvimento
- `pnpm build` — gera o build estático
- `pnpm preview` — serve o build localmente
- `pnpm deploy` — publica no GitHub Pages
